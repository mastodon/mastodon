= README

== Name

pkg-config

== Description

A pkg-config implementation by Ruby.

== Dependencies

* ruby >= 1.8 (1.9.2 is also supported!)

== Install

=== Package

  # gem install pkg-config

=== No package

  # ruby setup.rb
